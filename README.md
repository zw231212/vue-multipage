# vue-multipage

> A vue multi page project
> 多页面Vue项目
## 项目来源
[参考地址](https://juejin.im/post/5a8e3f00f265da4e747fc700#heading-1)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

##静态文件引用
static目录放外部的引用
assets目录放项目本身的引用
